Phase 0A — GPU Adam Optimizer

Adam is a gradient descent optimizer used in machine learning. It learns "how big to step" for each parameter by tracking momentum (average direction) and variance (average step size²). Here f(x,y)=x²+4y² — the oval bowl — and the dot finds the minimum at (0,0). Phase 0A implements this in WebGPU so the GPU can run 700 iterations per frame without sending data back to the CPU.

Phase 0B — SMPL Skeleton (Rodrigues FK)

SMPL is a 3D human body model with 24 joints. Rodrigues converts "rotate 0.3 rad around the X axis" (axis-angle) into a 3×3 rotation matrix. Forward Kinematics (FK) chains rotations from root (pelvis) to leaves (hands, feet). Phase 0B implements this in WebGPU WGSL and also computes the analytical derivative dR/dω needed for backpropagation.

Phase 0C — Loss Functions

Six loss functions penalise different types of "wrongness". L_p says "stay close to natural poses". L_smooth says "don't jitter between frames". L_3D says "the person should be where the depth sensor says". L_c1 says "feet should touch the floor". L_2D says "joints should project to where the camera saw them". Phase 0C implements all six as GPU compute shaders that run in parallel.

Phase 0D — Gradient Heatmap (JVP)

The chain rule converts "how much should vertex positions change?" into "how much should joint angles change?". This is the JVP (Jacobian-Vector Product). Phase 0D implements this as a WebGPU kernel that can compute all 72 gradients in parallel. Red = the optimizer needs to change this joint a lot; blue = this joint is already close to optimal.

Phase 0E — Optimizer Convergence

Phase 0E combines everything: the SMPL forward pass, all six loss functions, the JVP gradient, and Adam — and records all 700 iterations into a SINGLE GPU command. This means the CPU submits one batch job, the GPU works for ~10ms, then the CPU reads back the result. There is no back-and-forth between CPU and GPU during optimisation. The chart shows the loss curve: Stage 1 (iterations 0–500, lr=0.07) takes big steps, Stage 2 (500–700, lr=0.014) refines.
